What's the Difference ?
HTML

HyperText Markup Language

Create the structure

  • Controls the layout of the content

  • Provides structure for the webpage design

  • The fundamental building block of any web page
CSS

Cascading Style Sheet

Stylize the website

  • Applies style to the webpage

  • Targets various screen size to make web pages responsive

  • Primarily handles the "look and feel" of a webpage
Javascript

Increase interactivity

  • Adds interactivity to a web page

  • Handles complex functions and features

  • Programmatic code which enhances functionality
